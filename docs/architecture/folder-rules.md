# Folder Rules

- Do not use monorepo.
- Do not create `apps/`.
- Do not create `packages/`.
- Do not use workspace.
- Do not use pnpm.
- Do not use next@latest/react@latest/react-dom@latest.
- Use Node.js 24.x.
- UI components go to `src/components`.
- shadcn components go to `src/components/ui`.
- Business logic goes to `src/features`.
- API request goes through `src/lib/api/http-client.ts`.
- Learner must not use sidebar.
- Admin may use sidebar.
- Focus learning must not show main navigation.
